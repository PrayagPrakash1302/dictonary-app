# Dictionary App

A simple FastAPI + vanilla JS dictionary app that fetches meanings and synonyms.

## Features
- Search words
- Meanings list
- Synonyms list

## Tech Stack
- FastAPI (Python)
- Vanilla JS + HTML/CSS
- Dictionary API (dictionaryapi.dev)

## Setup
1. Create venv and install dependencies
2. Run backend
3. Open the app in the browser

## Run (Backend)
```powershell
cd backend
python -m uvicorn main:app --reload
